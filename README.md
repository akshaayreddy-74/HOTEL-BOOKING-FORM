# Hotel Booking Form

A simple and responsive Hotel Booking Form built using HTML, CSS, and JavaScript. This project allows users to enter booking details such as name, check-in date, check-out date, number of guests, and room preferences with basic validation.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- User-friendly booking form
- Basic form validation
- Responsive design
- Clean and simple UI

## How to Run
Open the `index.html` file in any web browser to use the hotel booking form.
